Platform Overview

BioPipeline is a comprehensive platform that streamlines the vaccine development process using advanced bioinformatics tools. It integrates various prediction and analysis tools into a single cohesive workflow, making it easier to identify potential vaccine candidates.

Key Features

  • Streamlined multi-step workflow from sequence analysis to final vaccine properties
  • Integration of industry-standard bioinformatics tools
  • Comprehensive result visualization and analysis
  • Downloadable reports and data at each step

Workflow Documentation

Step 1: FASTA Upload

The first step involves uploading a FASTA file containing protein sequences for analysis. The platform accepts standard FASTA format files with multiple sequences.

Step 2: Sequence Analysis

In this step, the uploaded sequences are analyzed using three specialized tools:

  • AlgPred: Predicts allergenicity of protein sequences
  • VaxiJen: Analyzes antigenicity of the sequences
  • Phobius: Predicts transmembrane topology and signal peptides

You can select a specific range of sequences for analysis and configure parameters for each tool. The results show how many sequences passed each analysis criteria.

Step 3: Epitope Prediction

This step predicts potential epitopes using four specialized tools:

  • ABCPred: B-cell epitope prediction
  • NetCTL: T-cell epitope prediction
  • NetMHC-I: MHC class I binding epitope prediction
  • NetMHC-II (Optional): MHC class II binding epitope prediction

Each tool can be configured with specific parameters. The results display the number of epitopes identified by each tool.

Step 4: Vaccine Properties

The identified epitopes are further analyzed for their vaccine potential:

  • Allergenicity: Confirms epitopes are non-allergenic
  • Antigenicity: Ensures epitopes are sufficiently antigenic
  • ToxinPred: Verifies epitopes are non-toxic

Results show how many epitopes passed each criterion and identify viable vaccine candidates.

Step 5: Final Results

The final step provides comprehensive analytics on the identified vaccine candidates:

  • Total number of viable epitopes
  • Estimated population coverage
  • Overall antigenicity and safety scores
  • Downloadable reports and data files

Tools Documentation

AlgPred

AlgPred is a tool for prediction of allergenic proteins and mapping of IgE epitopes. It uses various approaches such as support vector machines, motif-based method, and hybrid approaches.

Key Parameters:

  • Threshold Value: Determines the stringency of prediction (0-1)
  • Prediction Method: The approach used for prediction (hybrid, SVM, motif)

VaxiJen

VaxiJen is the first server for alignment-independent prediction of protective antigens. It uses an alignment-free approach based on auto cross covariance transformation of protein sequences into uniform vectors of principal amino acid properties.

Key Parameters:

  • Antigen Threshold: Cutoff value for antigen classification (0-1)
  • Target Organism: The type of pathogen being analyzed

Phobius

Phobius is a combined transmembrane topology and signal peptide predictor. It is used to identify transmembrane regions and signal peptides in proteins.

Key Parameters:

  • Prediction Method: Normal or short prediction mode
  • Output Format: Format of the prediction results

ABCPred

ABCPred is a tool for B-cell epitope prediction using artificial neural network. It predicts B-cell epitopes in an antigen sequence with accuracy.

Key Parameters:

  • Threshold Value: Cutoff for epitope prediction (0-1)
  • Epitope Length: Length of the epitope to be predicted

NetCTL

NetCTL predicts CTL (Cytotoxic T Lymphocyte) epitopes in protein sequences. The prediction method integrates MHC class I binding, TAP transport efficiency, and proteasomal cleavage.

Key Parameters:

  • MHC Supertype: MHC supertype to be used for prediction
  • Threshold Value: Cutoff for epitope identification

NetMHC

NetMHC predicts binding of peptides to MHC class I molecules. The method uses artificial neural networks (ANNs) trained on binding data.

Key Parameters:

  • MHC Allele: Specific MHC allele for binding prediction
  • Peptide Length: Length of peptides to be analyzed
  • Binding Threshold: Affinity cutoff in nM

ToxinPred

ToxinPred is a tool designed to predict and design toxic/non-toxic peptides. It uses machine learning approaches to predict toxicity.

Key Parameters:

  • Prediction Method: SVM or motif-based prediction
  • Toxicity Threshold: Cutoff for toxicity classification

Frequently Asked Questions

What file formats are accepted for sequence upload?

The platform accepts standard FASTA format files (.fasta, .fa, or .txt containing FASTA formatted sequences). Each sequence should have a header line starting with '>' followed by the sequence identifier.

How long does the analysis process take?

Processing time depends on the number and length of sequences, as well as the server load. Typically, each step takes from a few minutes to several hours for very large datasets.

Can I save my progress and return later?

Yes, the platform automatically saves your progress at each step. You can return to your analysis using the provided job ID.

What do the threshold values mean?

Threshold values determine the stringency of predictions. Higher thresholds usually result in fewer but more confident predictions, while lower thresholds identify more candidates but with potentially more false positives.

How can I interpret the results?

The platform provides detailed summaries and visualizations to help interpret results. Generally, sequences or epitopes that pass all criteria are considered the best vaccine candidates. Detailed explanation of each metric is provided in the documentation.
